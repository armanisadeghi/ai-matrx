## What this system is (30 seconds)

Every UI declares itself as a **surface**: a code-first manifest (`features/surfaces/manifests/*.manifest.ts`) declaring **surface values** (named runtime state the page can supply), mirrored into `ui_surface` / `ui_surface_value` by manifest sync. Pages emit live values via a `buildScope()` builder + `UnifiedAgentContextMenu surfaceName=...`. Agents consume values through scoped **bindings** (`agx_agent_surface.value_mappings`, mapTypes `surface_value | direct_value | prompt_user | unmapped`) and **shortcuts** (`agx_shortcut`), resolved per key at launch. Phase 3 added **agent roles** (surfaces declare agent positions; users/orgs override the default per role) and **centralized scoped per-surface config** (`ui_surface_config`, namespaced JSONB) so features stop building private settings tables.

The reference implementation end to end is **`matrx-user/transcripts-cleanup`** (`/transcripts/cleanup`, `features/transcription-cleanup/`): 36 values, full live emitter, 2 agent roles, dictionary namespace declared.
